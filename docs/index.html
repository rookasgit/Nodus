Custom Implementation

Fuzzy Anchoring.

Our search algorithm ignores whitespace and punctuation shifts to ensure margin notes survive your stylistic refactoring.

Five-step custom algorithm — no external dependency. Exact match first, then character-sequence regex with whitespace and punctuation injected between every character. A 2.5× length guard prevents runaway matches across paragraphs if the user deletes surrounding text.

// 1. Try exact match first let idx = text.indexOf(quote); // 2. If not found, build whitespace-agnostic regex if (idx === -1) { const chars = quote.replace(/[\s\W_]+/g, '').toLowerCase(); const pattern = chars.split('').join('[\\s\\W_]*'); const match = new RegExp(pattern, 'i').exec(text); // 3. Reject if match spans > 2.5x original length (runaway guard) if (match && match[0].length < quote.length * 2.5) { idx=match.index; } }
